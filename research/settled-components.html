The Quote Form's Settled Language

Living component sheet harvested from the redesigned quote form — the locked design foundation. Build new pages by lifting these surfaces. Narrative spec + the 10 Rules: settled-components.md.

Core law: blue = action · orange = the wink (fills, chips, Orange-900 large accents — never #EF8B22 text) · primary text = Navy 500 · white cards on light warm backgrounds · illustration not photography · trust is ambient · motion is calm (120/180/240ms) · everything on the 4px grid.

Updated 6/11 PM — KP design-system refinements (Emma Butler) adopted: tonal ramps + WCAG pairings, 20px-base Size 1–6 type scale (Lora = banner only, Bebas retired from website work), 4px spacing system, iconography rules. The Rules are now 12 (see settled-components.md).

The rules: every gap is a token — never 15/22/35px. Card internal padding 40px (32 tablet, 24 mobile); card-to-card gap 32px = the gutter. Section rhythm 112 / 80 / 64 (desktop → tablet → mobile, step down ONE level per breakpoint). Heading→body 20px; paragraph spacing 32px (practical rhythm unit for the 30px line height). Form fields 56px tall, 24px apart. Touch targets never below 44px. Proximity signals relationship; nested spacing decreases; shrink spacing before shrinking type.

Iconography — one SVG set, orange fill → blue hover (KP refinements, adopted 6/11)

Einstein-iconography-rules.md · agent/visual/icons.md (hover the trucks)

Three sizes only: 40 / 56 / 80px, all with 16px padding, 8px icon-to-label gap, viewBox="0 0 24 24". Primary fill = Orange-500 → Blue-500 on hover/focus → Blue-700 pressed → Grey-200 @40% disabled, animated fill 150ms ease-in-out. Orange-500 is legal here because it's a FILL — the same hex as text isn't. One consistent SVG set sitewide (no mixed libraries, no icon fonts); stroke weight scales with size (1.5/2/2.5px). The emoji you see across this sheet are a placeholder convention for this workshop — explicitly NOT for production. Production icons come from the kit's orange/navy line set.

The wink survives, the hex changes. Small winks (jokes, weekday nudges — never surge framing, "why we ask" helpers): italic Navy-500 text on an Orange-50 #FEF6EC chip, rounded, subtle, one emoji max. Classes: .nudge (chip) and .nudge-display.
